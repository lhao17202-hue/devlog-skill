# 更新日志

所有重要的变更记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] — 2026-07-12

### 新增
- `setup.py` — 项目初始化脚本，创建 `.devlog/` 目录和配置文件
- `log_session.py` — 会话日志记录脚本，支持结构化会话数据录入
- `generate_report.py` — 报告生成脚本，支持日报/周报/迭代方案三种类型
- 日报：今日概览 + 完成事项 + 卡点 + 基于数据的明日计划
- 周报：进度趋势 + 错误分类统计 + 文件热点分析 + 迭代建议
- 迭代方案：根因分析 + 工时估算 + 按天排期 + 风险矩阵 + 成功标准 + 改进闭环
- 项目进度索引 (`index.json`) 自动维护
- 日志 JSON Schema 文档
- 报告模板参考文件
- 中英文双语支持
